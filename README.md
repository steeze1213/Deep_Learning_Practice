# Deep Learning Practice

개인적으로 머신러닝, 딥러닝을 독학하며 실습하며 작성한 코드 모음입니다.
Google Colab / Jupyter Notebook 환경에서 실습한 노트북을 정리한 개인 학습 기록용 레포지토리입니다.

---

## 사용 교재

혼자 공부하는 머신러닝 + 딥러닝
- 출판 연도: 2020

---

## 학습 내용 요약

### Chapter 01 – 머신러닝·딥러닝 개요 및 개발 환경
- 인공지능/머신러닝/딥러닝 개념, Colab·Jupyter 사용

### Chapter 02 – 데이터 다루기와 전처리
- 훈련/테스트 분리, 데이터 전처리, 스케일링

### Chapter 03 – 회귀 알고리즘과 모델 규제
- KNN 회귀, 선형/다항 회귀, 규제(Ridge/Lasso)

### Chapter 04 – 분류 알고리즘
- 로지스틱 회귀, 확률적 경사 하강법(SGD)

### Chapter 05 – 트리 기반 모델과 앙상블
- 결정 트리, 교차 검증, 랜덤 포레스트, 부스팅

### Chapter 06 – 비지도 학습
- 군집화(K-means), 차원 축소(PCA)

### Chapter 07 – 신경망과 딥러닝 기초
- 인공 신경망, 심층 신경망, 모델 훈련/정규화

### Chapter 08 – 이미지 딥러닝 (CNN)
- 합성곱 신경망, 이미지 분류, CNN 시각화

### Chapter 09 – 텍스트 딥러닝 (RNN/LSTM/GRU)
- 순환 신경망, IMDB 감성 분류, LSTM/GRU

### Chapter 10 – 트랜스포머와 언어 모델
- 어텐션, 트랜스포머, 사전학습 언어 모델, LLM 활용
